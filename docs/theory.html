Theory¶

This chapter provides some theoretical background to the larvotto project.

On the minimum image convention¶

This is a 1D derivation. We consider the simulation box \((0,L)\), and two particles, located at \(x\) and \(y\). We seek the radius-vector \(r\) in the minimum image convention, defined as follows

(1)¶\[r=y-x+nL, \quad\text{where}\quad d(x, y)=\arg\min\bigl\{\lvert y-x+mL\rvert,m\in\mathbb Z\bigr\}.\]

In this section, we derive the closed-form expression of this distance. We first observe that

\[\lvert y-x\rvert\leq L, \quad\text{since}\quad 0\leq y\leq L \quad\text{and}\quad -L\leq -x\leq 0.\]

Without loss of generality, we can assume that \(x\leq y\), so that

(2)¶\[\lvert y-x\rvert=y-x\quad\text{and}\quad0\leq y-x\leq L.\]

Under the above assumption, \(n\) in Eq. (1) must be 0 or -1. Indeed, it results from Eq. (2) that

(3)¶\[nL\leq y-x+nL\leq\bigl(n+1\bigr)L.\]

It results from the above inequality

\begin{align} \text{for }n\geq 1:&& y-x+nL&\geq nL\geq L\\ \text{for }n\geq-2:&& y-x+nL&\leq\bigl(n+1\bigr)L\leq-L \end{align}

and, in both cases

\[\lvert y-x+nL\rvert\geq L\geq\lvert y-x\rvert\]

The minimum is necessarily reached for \(n>1\) and \(n >-2\); in other words, the value of \(n\) in Eq. (1) is 0 or -1. In what follows, we discuss under which conditions we have \(n=-1\).

For \(n\) to be equal to \(-1\), we must have

(4)¶\[\lvert y-x-L\rvert<y-x.\]

From Eq. (3), we again observe that

(5)¶\[y-x-L\leq 0\quad\text{therefore}\quad\lvert y-x-L\rvert=L+x-y.\]

Combining Eqs. (4) and (5), we find the condition

\[L+x-y<y-x,\quad\text{therefore}\quad 2\bigl(y-x\bigr)>L.\]

To sum up, we have shown that

\[\begin{split}\text{for}\quad x\leq y:\quad n= \begin{cases} 0 & \text{if}\quad2r_0\leq L\\ -1 & \text{if}\quad2r_0>L \end{cases} \quad\text{with}\quad r_0=y-x.\end{split}\]

When \(x>y\), the above result applies if we swap \(x\) and \(y\). Therefore, minimum image radius-vector \(r\) of any \(0\leq x,y\leq L\) is given by the following expression

\[\begin{split}r= \begin{cases} r_0-L & \text{if}\quad2r_0>L\\ r_0+L & \text{if}\quad2r_0<-L\\ r_0 & \text{otherwise} \end{cases} \quad\text{with}\quad r_0=\lvert y-x\rvert.\end{split}\]
