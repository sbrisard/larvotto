Vector minimum_image(const Vector x, const Vector y) const¶

Return the minimum image r of the difference r₀ = y - x.

This vector is defined as follows 

rᵢ = r₀ᵢ + nᵢ × Lᵢ,
where 
nᵢ = arg min {|r₀ᵢ + m × Lᵢ|, m = …, -2, -1, 0, 1, 2, …}.
